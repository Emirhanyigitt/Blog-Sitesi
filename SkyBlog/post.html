Temel Terminal Komutları


Files & Navigating


pwd: Hangi konumda olduğumuzu belirtir.

mkdir “klasör”: Dosya oluşturmamızı sağlayan komuttur.

ls: Olduğunuz dosyayı listeler.

ls -force: ls komutundan farkı gizli dosyalarıda göstermesi.

cd. : Bulunduğunuz klasöre gider.
cd./.. : Bulunduğunuz klasörün bir üstüne gider.
cd.. : Bulunduğunuz klasörün bir üstüne gider.
cd../klasor : Bir üstündeki belirttiğiniz klasöre gider.

rm “klasor” : Belirttiğiniz dosyayı siler.

rm -f dir : Dosyayı zorlayarak kaldırır.

rm -r dir : Dosyayı zorlayarak siler.

rm -rf dir : Dosyayı kaldırmaya yarayan komuttur.

rm -rf/ : Her şeyi siler.

cp klasor1 klasor2 : Seçtiğiniz iki dosyayı kopyalar.

mv klasor1 dir/klasor2 : Seçtiğiniz iki dosyanında ismini yeniden adlandırır.

touch file : Yeni dosyalar oluşturursunuz.

cat file : Belirttiğiniz dosyanın içeriğini görüntüler.

cat > file : Dosyanın içine metin koymamızı sağlar.

cat >> file : Çıktınızı dosyaya ekler.

tail -file : Dosyanın içeriğini çıktı olarak gösterir.


Networking


ping host : Ping attığınız sunucu.

whois domain : Alan adının kimin olduğunu söyleyen komut.

dig domain : Dns almayı sağlar.

wget file : Telnet ağı komut satırı ile dosya indirmeyi sağlar.

wget -c file : Dosya indirmeyi durdurulduğunda devam ettirir.

wget -r url : Url den tekrardan dosya indirmeyi sağlar.


Processes


ps : Aktif olan işlemleri gösterir.

ps aux : Ayrıntılı açıklamasını sağlar.

kill pid : Tüm süreçleri yok eder.

killall proc : Proc isimli tüm işlemleri yok eder.


System Info


date : Tarihi ve saati gösterir.

uptime : Çalışma süresini gösterir.

whoami : Kim olduğunu söyler.

w : Kimin çevrimiçi olduğunu gösterir.

cat/proc/cpuinfo : İşlemci bilgisini gösterir.

cat/proc/meminfo : Hafıza bilgisini gösterir.


Compressing


tar cf file.tar : Files tar dosyaları gösterir.

tar xf file.tar : Geçerli dizine yayılmış olarak gösterir.


Permissions


chmod actol file : Dosya izinlerini değiştirir.

chmod 777 : Herkes için rewx.

chmod 755 : Sahip için rw grup için rx.


Some Others


grep pattern files : Desen için dosya arama.

grep -r pattern dir : Dizinde yinelemeli olarak dizin arama.

locate file : Tüm dosya örneklerini bulur.

whereis app : Uygulamanın geçerli konumunu gösterir.

man command : Komutlar için kılavuz gösterir.

Bu makelemde Temel Terminal Komutlarını kısaca anlattım, okuduğunuz için teşekkürler.
